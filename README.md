## 주요 명령어

### 설치

```
git clone https://github.com/Block-Chain-Valley/bv-contract-boilerplate.git
```

### 컨트랙트 빌드

```
yarn build
```

### 컨트랙트 테스트

```
# 테스트 결과 출력
yarn test

# 테스트 결과 및 커버리지 결과 출력
yarn coverage
```
